- `getPathWithoutLocale` — 去除路径中的语言前缀

|------|------|----------|------|
| 患者门户 | `/[lang]/portal/patient` | `app.../login?role=patient` | ✅ 已完成 |

| 管理门户 | `/[lang]/portal/admin` | `app.../login?role=admin` | ✅ 已完成 |

每页包含：角色标题、登录提示文案、「去登录」按钮、「返回首页」链接。

## 未完成功能 / 已知限制

以下能力 **不在 v0.1.0 范围内**，或当前仅为占位：

| 项目 | 说明 |
|------|------|
| 后端 API | 无 `app/api/` 目录，无任何 Route Handler |
| 数据库 | 无 ORM、无数据模型、无持久化层 |
| 用户认证 | 登录逻辑在 App 子域，本仓库仅读取 Cookie 状态 |
| 门户实际功能 | 门户页仅为跳转引导，无业务界面 |
| 联系表单 | 联系页无表单、无邮件地址、无提交 API |
| GitHub 链接 | 当前指向 `https://github.com` 占位地址 |
| CMS | 项目列表、统计数据均为硬编码，无法动态更新 |
| 测试 | 无单元测试、E2E 或 CI 配置 |
| Analytics | 无 Google Analytics、Sentry 等监控集成 |
