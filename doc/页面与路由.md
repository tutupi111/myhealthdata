# 页面与路由

**文档最后更新**：2026-05-21 23:10:06（UTC+8）

## 路由总览

| 路由 | 页面 | 渲染方式 | 说明 |
|------|------|----------|------|
| `/` | 重定向 | 静态 | 自动跳转至 `/en` |
| `/en` / `/zh` | 首页 | SSG | HealthData 深色主题落地页 |
| `/[lang]/mission` | 使命页 | SSG | 基金会使命说明 |
| `/[lang]/projects` | 项目页 | SSG | 项目列表展示 |
| `/[lang]/research` | 研究页 | SSG | 研究 / 应用场景简介 |
| `/[lang]/join` | 加入页 | SSG | 加入引导 |
| `/[lang]/contact` | 联系页 | SSG | 邮箱 + 留言表单 |
| `/[lang]/portal/patient` | 个人门户 | SSG | 跳转 App 登录 |
| `/[lang]/portal/researcher` | 研究者门户 | SSG | 跳转 App 登录 |
| `/[lang]/portal/admin` | 管理门户 | SSG | 跳转 App 登录 |
| `/api/contact-events` | 联系事件 API | 动态 | 代理至 Data Hub |
| `/sitemap.xml` | 站点地图 | 动态生成 | SEO |
| `/robots.txt` | 爬虫规则 | 动态生成 | SEO |

## 导航结构（v0.2.0）

Navbar 深色栏，路径均带语言前缀 `/{lang}`：

| 导航项 | 路由 | 说明 |
|--------|------|------|
| Home | `/[lang]` | 首页 |
| Platform | `/[lang]/projects` | 项目 / 平台 |
| Use Cases | `/[lang]/research` | 研究 / 应用场景 |
| Docs | `https://github.com` | 外链（占位） |
| About | `/[lang]/mission` | 使命 / 关于 |
| Open Source | `/[lang]/#open-source` | 首页开源锚点 |

右侧操作区：

- 语言切换（en ↔ zh）
- 登录 / 进入管理中心（Cookie 状态）
- **Join the Movement** → `/[lang]/join`

## 首页区块结构

```
/[lang]/
├── HeroSection              标签 + 标题 + CTA + 3D 插画
├── 健康数据问题（4 ProblemCard）     浅色背景
├── 愿景 + 全球影响（4 GlobalImpactCard）  深色背景
├── 我们的工作（3 FeatureCard）       浅色背景
├── 平台入口（3 PortalCard）
│   ├── 个人 → /[lang]/portal/patient
│   ├── 研究者 → /[lang]/portal/researcher
│   └── 组织机构 → /[lang]/contact
├── 精选项目（3 ProjectCard）         浅色背景
├── 开源 #open-source                  深色背景
└── Join the Movement CTA             渐变背景
```

## 门户与 App 跳转

| 入口 | 目标 |
|------|------|
| 个人门户 | `{APP_BASE_URL}/login?role=patient` |
| 研究者门户 | `{APP_BASE_URL}/login?role=researcher` |
| 管理门户 | `{APP_BASE_URL}/login?role=admin` |
| 登录 / 管理中心 | Navbar 根据 Cookie 跳转 App |
