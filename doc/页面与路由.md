# 页面与路由

## 路由总览

| 路由 | 页面 | 渲染方式 | 说明 |
|------|------|----------|------|
| `/` | 重定向 | 静态 | 自动跳转至 `/en` |
| `/en` | 英文首页 | SSG | 完整营销落地页 |
| `/zh` | 中文首页 | SSG | 完整营销落地页 |
| `/[lang]/mission` | 使命页 | SSG | 基金会使命说明 |
| `/[lang]/projects` | 项目页 | SSG | 项目列表展示 |
| `/[lang]/research` | 研究页 | SSG | 研究简介 |
| `/[lang]/join` | 加入页 | SSG | 加入引导 |
| `/[lang]/contact` | 联系页 | SSG | 联系简介 |
| `/[lang]/portal/patient` | 患者门户 | SSG | 跳转 App 登录 |
| `/[lang]/portal/researcher` | 研究者门户 | SSG | 跳转 App 登录 |
| `/[lang]/portal/admin` | 管理门户 | SSG | 跳转 App 登录 |
| `/sitemap.xml` | 站点地图 | 动态生成 | SEO |
| `/robots.txt` | 爬虫规则 | 动态生成 | SEO |

`[lang]` 有效值为 `en` 或 `zh`。

## 导航结构

Navbar 包含以下链接（路径均带语言前缀）：

```
首页 (/)          → /[lang]
使命 (/mission)   → /[lang]/mission
项目 (/projects)  → /[lang]/projects
研究 (/research)  → /[lang]/research
加入 (/join)      → /[lang]/join
联系 (/contact)   → /[lang]/contact
```

右侧操作区：

- 语言切换（en ↔ zh）
- 登录 / 进入管理中心（取决于 Cookie 状态）

## 门户页跳转逻辑

门户页不承载业务功能，点击「去登录」后跳转至 App 子域：

| 门户 | App 登录 URL |
|------|--------------|
| 患者 | `{APP_BASE_URL}/login?role=patient` |
| 研究者 | `{APP_BASE_URL}/login?role=researcher` |
| 管理 | `{APP_BASE_URL}/login?role=admin` |

`APP_BASE_URL` 默认为 `https://app.myhealthdata.foundation`，可通过环境变量 `NEXT_PUBLIC_APP_URL` 覆盖。

## Sitemap 优先级

| 路径类型 | priority |
|----------|----------|
| 首页 `/[lang]` | 1.0 |
| 内容页（mission、projects 等） | 0.8 |
| 门户页 | 0.6 |
