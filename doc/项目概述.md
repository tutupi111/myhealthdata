# 项目概述

**文档最后更新**：2026-05-21 23:10:06（UTC+8）

## 项目名称与定位

**myhealthdata** 是 MyDataHealth Foundation 的官方网站（前端品牌展示为 **HealthData**），面向公众宣传「健康数据属于每个人」的核心理念，展示基金会使命、项目与平台入口，并将用户引导至独立应用子域完成登录与业务操作。

| 项目 | 说明 |
|------|------|
| 名称 | myhealthdata |
| 版本 | 0.2.0 |
| 品牌名 | HealthData |
| 类型 | Next.js 营销官网 + 联系表单 API 代理 |
| 生产域名 | https://www.myhealthdata.foundation |
| 联系邮箱 | hugchen1117@gmail.com |
| 应用子域 | https://app.myhealthdata.foundation |
| 许可证 | MIT |

## 技术栈

| 层级 | 技术 |
|------|------|
| 框架 | Next.js 16.1.6（App Router） |
| UI | React 19.2.3 + TypeScript 5 |
| 样式 | Tailwind CSS 4 |
| 字体 | Google Inter（`next/font`） |
| 部署 | Vercel |
| 国际化 | 自研静态翻译（`lib/i18n/translations.ts`），支持英文 / 中文 |
| 联系事件 | App Data Hub（`https://tuutpi.online`） |

## 双站点架构

```
┌─────────────────────────────────────┐
│  www.myhealthdata.foundation        │  ← 本仓库
│  营销官网 / 双语静态页 / 联系表单     │
│  读取 Cookie: mhdf_logged_in        │
│  POST /api/contact-events → Data Hub│
└──────────────┬──────────────────────┘
               │ 外链登录 / 管理中心
               ▼
┌─────────────────────────────────────┐
│  app.myhealthdata.foundation        │  ← 独立项目
│  登录 / 仪表盘 / 实际业务功能        │
│  写入 Cookie: mhdf_logged_in        │
└─────────────────────────────────────┘
```

## 当前版本范围（v0.2.0）

**已完成**：

- 双语页面、HealthData 深色主题首页、SEO、响应式布局
- 联系页（邮箱 + 表单）与 Data Hub 事件对接
- 门户引导页、App 子域登录 Cookie 联动

**不在本仓库范围内**：

- 用户注册/登录实现、医疗数据存储与处理
- CMS 内容管理、Analytics 集成
- 真实 GitHub 仓库地址（当前仍为占位链接）

变更历史见 [更新日志](./更新日志.md)。
