## 版本信息

- **当前版本**：0.1.0
- **项目类型**：MyDataHealth Foundation 双语营销官网（静态展示站）
- **生产域名**：https://www.myhealthdata.foundation
- **关联应用**：https://app.myhealthdata.foundation（独立项目，不在本仓库）

## 重要说明

本仓库 **不包含** 后端 API、数据库、用户认证或医疗数据处理逻辑。实际业务平台（登录、仪表盘、数据管理）部署在 `app.myhealthdata.foundation` 子域；本仓库负责品牌展示、内容传播，并将用户引导至外部 App。
